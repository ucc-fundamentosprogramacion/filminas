C

Lenguaje de programación de tipo estructurado, cuyo enfoque de solución de los problemas se disena de arriba hacia abajo (top-down), y de izquierda a derecha (left to right).

/*Programa 1.1
Imprime un mensaje por pantalla*/

C: Primeros Pasos

La función main es de donde se comienza a ejecutar el programa principal.

El primer void indica que el programa no arrojará resultados de un tipo de datos.

El segundo void especifica que el programa no tiene parámetros.

Todas las intrsucciones tienen que estar dentro del bloque {} y finalizar con ;, excepto si son estructuras selectivas o repetitivas.

C: Formato de Variables

El formato de lectura y escritura de variables cambia de acuerdo al tipo de datos que estas puedan tener.

La especificación de formato es obligatora al escribir instrucciones de lectura scanf y de escritura printf

C: Formato de Variables

Formato Explicación
%u Escribe enteros sin signo de 2 bytes (unsigned int)
%d %i Escribe enteros 2 bytes (int)
/ld Imprime enteros de largo alcance (long)
%f Escribe reales de 4 bytes (float)
%lf Escribe reales de doble precisión, 8 bytes (double)
%e Imprime en forma exponencial
%g Imprime en %f o %e en función del tamano del número.
%c Escribe un caracter de byte (char)
%s Escribe una cadena de caracteres, que termina con '\0'

C: Formato de Variables

float x=6.2555, z=7.2576;
int y=4, t=-5;

printf("%f %d %f %d", x,y,z,t);
printf("%f \n %d \t %f \n %d", x,y,z,t);
Produce la siguiente salida: 6.255500 4 7.257600 -5

C: Modificadores de Símbolos

Formato Explicación
%5d Escribe un entero utilizando un campo de 5 dígitos. Justificación a la derecha.
%-6d Escribe enteros utilizando un campo de 6 dígitos. Justificación a la izquierda.
%4.2f Escribe un real utilizando un campo de 4 dígitos. 2 de ellos son decimales.
%-5.2f Escribe un real utilizando un campo de 4 dígitos. 2 de ellos decimales. Justificación a la izquierda.

C: Modificadores de Símbolos

float x=6.2555, z=7.2576;
int y=4, t=-5;

printf("%4.2f \n %5.2e \n %5d \n %d", x,z,y,t);
Produce la siguiente salida: 6.25 7.26e+00 4 -5

Contador

Es una variable que se utiliza para contar.

Es una variable que se incrementa de una unidad cada vez que se repite cierta acción.

En programación, la expresión X = X+1 se diagrama como X <- X+1

Cuando se trabaja con contadores, siempre se les debe asignar un valor inicial (inicializar). ((te animas a hacer la prueba de que pasa cuando no inicializas el contador??))

Contador: Ejemplo

Asignación Valor de X
X <- 0 0
X <- 1 1
X <- 2 2
X <- 3 3

Contador

A la izquierda de una sentencia de asignación sólo puede figurar una constante ó variable.

Lo que está a la derecha de la sentencia de asignación se valúa y el valor obtenido se le asigna a la constante o variable que figura a la izquierda de la misma.

Sumador

Es una variable que se utiliza para ir sumando o acumulando datos.

Cuando se trabaja con sumadores, siempre se les debe asignar un valor inicial (inicializar).

Después de inicializados, se van sumando los valores deseados.

Sumador: Ejemplo

Asignación Valor de SUMA
Suma <- 0 0
Suma <- 20 20
Suma <- 30 50
Suma <- 10 60

Programa

Todo programa tiene un nombre que lo define y debe ser representativo de la función que cumplen.

Es recomendable escribir un párrafo como comentario de lo que realiza el programa.

También se pueden definir las constantes y variables que utilizaremos en el desarrollo de la solución.

Programa

/*NOMBRE DEL PROGRAMA
* Descripción de que hace
* Version: 2.1
* Fecha: 04/03/2019
*/

    #include <stdio.h>
#include <stdlib.h>

int main() {
    //Comentario en línea independiente. Declaración de la variable edad
    int edad;
    edad=54; //Comentario en línea junto a código. Asignación de valor a la variable edad
    printf ("La edad es %d\n", edad);
    printf ("Gracias por utilizar este programa");
    /* Finalizamos el programa, comentario en bloque en una sola línea */
    return 0; /*Sentencia return de terminación, comentario en bloque en una sola línea*/
}
